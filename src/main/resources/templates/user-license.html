1. Общие положения

1.1. Minecraft Union (далее — «MU») предлагает пользователю сети Интернет (далее – Пользователь) - использовать свои развлекательные ресурсы на условиях, изложенных в настоящем Пользовательском соглашении (далее — «Соглашение», «ПС»). Соглашение вступает в силу с момента выражения Пользователем согласия с его условиями в порядке, предусмотренном п. 1.4 Соглашения.

1.2. MU предлагает Пользователям доступ к широкому спектру медиа- и развлекательных ресурсов, а так же игровых площадок. Все существующие на данный момент интернет-ресурсы, а также любое развитие их и/или добавление новых является предметом настоящего Соглашения.

1.3. Использование интернет-ресурсов MU регулируется настоящим Соглашением, а также условиями посещения отдельных интернет-ресурсов. Соглашение может быть изменено MU без какого-либо специального уведомления, новая редакция Соглашения вступает в силу с момента ее размещения в сети Интернет по указанному в настоящем абзаце адресу, если иное не предусмотрено новой редакцией Соглашения. Действующая редакция ПС всегда находится на странице по адресу https://mwmw.icu/user-license.

1.4. Начиная использовать какой-либо интернет-ресурс/его отдельные функции, либо пройдя процедуру регистрации, Пользователь считается принявшим условия Соглашения в полном объеме, без всяких оговорок и исключений. В случае несогласия Пользователя с какими-либо из положений Соглашения, Пользователь не вправе использовать интернет-ресурсы MU. В случае если MU были внесены какие-либо изменения в Соглашение в порядке, предусмотренном пунктом 1.3 Соглашения, с которыми Пользователь не согласен, он обязан прекратить использование интернет-ресурсов MU.

2.1. Для того чтобы воспользоваться некоторыми интернет-ресурсами MU или некоторыми отдельными функциями интернет-ресурсов, Пользователю необходимо пройти процедуру регистрации, в результате которой для Пользователя будет создана уникальная учетная запись.

2.2. Для регистрации Пользователь обязуется предоставить достоверную и полную информацию о себе по вопросам, предлагаемым в форме регистрации, и поддерживать эту информацию в актуальном состоянии. 3.1. При регистрации Пользователь самостоятельно выбирает себе логин (уникальное символьное имя учетной записи Пользователя) и пароль для доступа к учетной записи. 5.1. Одновременно с завершением процесса регистрации учетной записи Пользователя, для него генерируется Индивидуальный Счет (далее - «ИС», «Счет»). Счет предназначен для совершения внутриигровых приобретений на интернет-ресурсах MU.

6.3. MU вправе редактировать Правила без особых уведомлений Пользователя. Новые редакции Правил находятся по указанным в П. 6.2 настоящего Соглашения адресам в сети Интернет.
